# Mascot Asset Slots

The GUI uses the prepared image files in this folder. To replace artwork while
keeping the GUI code unchanged, keep the same file names:

- `idle.png`
- `loading.png`
- `success.png`
- `error.png`
- `helper.png`
- `left_corner.png`
- `right_corner_blue.png`
- `preprocess_welcome.png`
- `recognition_welcome.png`
- `camera_welcome.png`
- `result_welcome.png`

Do not commit copied third-party character artwork unless the team has permission
to use it in the course project.
